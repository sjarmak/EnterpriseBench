# Support Ticket: Chat app goes blank when clicking threads button

**Priority:** Medium
**Submitted by:** Team Admin
**Product:** Element Web (self-hosted)

---

Hello,

Several people on our team are reporting that Element goes completely white (blank screen) when they click the threads button in the top-right corner of certain chat rooms. It doesn't happen in every room -- just some of them.

When it happens, the whole chat interface disappears and you're left with a white page. You have to refresh the browser to get back to the app, and if you go back to the same room and click threads again, it crashes again.

The rooms where this happens seem to be older rooms that were created before we enabled threading. Newer rooms where threads have been used from the start work fine.

We're on Element Web, self-hosted. We haven't updated recently. This is disrupting our team communication because people are afraid to click the threads button.

Can you look into what's happening when the threads button is clicked? We need to understand the code path so our dev team can investigate further.
