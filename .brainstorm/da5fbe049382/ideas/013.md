# Idea 13: Dead Code and Feature Flag Necropsy

Large OSS projects accumulate dead code, abandoned feature flags, and unreachable branches. Take projects known to have done major dead-code cleanup PRs (React, Angular, VS Code have all done this). Present the codebase BEFORE the cleanup. Task: identify dead code with confidence scores. Ground truth: the actual cleanup PR. Unique structure: this is an ABSENCE detection task — the agent must prove that something is NOT reachable, which requires complete call graph understanding. Feature flags add another layer: code that's syntactically reachable but semantically dead because the flag is permanently off. This is enterprise maintenance 101.
